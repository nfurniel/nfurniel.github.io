Sobre Mí

Mi Historia

Soy Nicolas Furnieles, estudiante de segundo año de Desarrollo de Aplicaciones Web y Escritorio (DAWE) en la UTAD. Mi pasión por la tecnología y el desarrollo web comenzó cuando descubrí que podía crear experiencias interactivas que las personas pueden usar y disfrutar.

Durante mi formación en la UTAD, he aprendido a trabajar con una amplia variedad de tecnologías modernas. Desde el frontend con HTML, CSS, JavaScript, React y frameworks como Bootstrap y Tailwind CSS, hasta el backend con PHP, Laravel y bases de datos como MySQL y MongoDB. También he profundizado en TypeScript y JavaScript moderno para escribir código más robusto y mantenible.

Me encanta el proceso completo de desarrollo: desde diseñar la interfaz de usuario hasta implementar la lógica del servidor y gestionar bases de datos. Cada proyecto es una oportunidad para aprender algo nuevo y mejorar mis habilidades. Estoy constantemente explorando nuevas tecnologías y mejores prácticas para convertirme en un desarrollador más completo.

Mi Filosofía

Aprendizaje Continuo

La tecnología evoluciona constantemente. Me comprometo a mantenerme actualizado con las últimas tendencias y mejores prácticas del desarrollo web.

Código Limpio

Escribir código legible y mantenible es fundamental. Me esfuerzo por seguir las mejores prácticas y convenciones de cada tecnología que utilizo.

Experiencia de Usuario
